# Login Bilgileri

Kullanıcı zetacad programına login olurken yetkili olduğu proje firmalarından birini seçmelidir. Eğer sadece bir bölgede yetkili ise sadece yetkili olduğu bölgeyi seçecektir. 


**Login Paneli:**

![login](../../assets/resimler/login.png) 


Login olduktan sonra **Ayarlar / Firma Bilgileri** panelinden, Login olunan;


- **kullanıcı bilgileri**  detayları,
- **firma bilgileri**  detayları

görünecektir.

# Genel Bakış

**Genel Bakış**  



1. Kurulum Adımları (Step by Step)
## Kurulum

1. Programı indirin
2. Kurulum dosyasını çalıştırın
3. Lisans anahtarınızı girin
4. Programı başlatın


Numaralı adımlar listesi.

2. Önemli Uyarı
!!! warning "Önemli"

    Programı kaldırmadan önce veritabanı yedeği alın.


Turuncu warning kutusu

3. Bilgi Kutusu
!!! info "Bilgi"

    Bu özellik yalnızca yönetici kullanıcılar için aktiftir.


Mavi bilgi kutusu.

4. İpucu
!!! tip "İpucu"

    Ctrl + S kısayolu ile projeyi hızlıca kaydedebilirsiniz.


Yeşil ipucu kutusu.

5. Açılır Bilgi (Detay gizleme)
??? note "Detayları göster"

    Sistem burada ek teknik kontroller yapar.


Kapalı bir kutu → tıklanınca açılır.

6. Varsayılan Açık Açılır Kutu
???+ note "Teknik detay"

    Program yaklaşık 400 farklı kontrol noktası üzerinden projeyi analiz eder.


Açık gelen expandable kutu.

7. Kod Örneği
```sql
SELECT * FROM Projects
WHERE Status = 'Active'
```


Syntax highlight olan kod bloğu.

8. Komut Satırı Örneği
```bash
mkdocs serve
```


Terminal komutu.

9. Sekmeli İçerik
=== "Windows"

    Programı `.exe` dosyasından çalıştırın.

=== "Linux"

    Terminalden şu komutu kullanın:

    ```bash
    ./program
    ```


Sekmeler:

Windows | Linux

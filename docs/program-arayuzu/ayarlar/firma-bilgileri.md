# Firma Bilgileri

Kullanıcı zetacad programına [giriş](../arayuz-elemanlari/giris.md) yaparken yetkili olduğu proje firmalarından birini seçmelidir. Eğer sadece bir bölgede yetkili ise sadece yetkili olduğu bölgeyi seçecektir. 

Bu bilgileri görmek için **Ayarlar / Firma Bilgileri** paneli açılabilir. 


- **kullanıcı bilgileri** 
- **firma bilgileri** 

dipostan otomatik getirilmektedir. 


![firmaBil](../../assets/resimler/firmaBil.png)   
  

???+ note "Hızlı Firma Değiştirme"
    __Birden fazla firmada yetkili iseniz__ <br>Bu paneldeki **FİRMA** alanından firmalarınız arasında hızlı değişim yapabilirsiniz.

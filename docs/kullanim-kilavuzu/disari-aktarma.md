# Dışa Aktarma

ZetaCAD projenizi **DXF** veya **PDF** formatında dışa aktarabilirsiniz.

## DXF Olarak Dışa Aktarma

DXF formatı, AutoCAD ve diğer CAD programlarıyla uyumludur.

1. Menü çubuğundan **Dosya → Dışa Aktar → DXF** seçin
2. Kayıt konumunu ve dosya adını belirleyin
3. **Kaydet**'e tıklayın

!!! info "DXF Uyumluluğu"
    ZetaCAD, AutoCAD 2010 ve üzeri sürümlerle uyumlu DXF formatı üretir.

## PDF Olarak Dışa Aktarma

1. Menü çubuğundan **Dosya → Dışa Aktar → PDF** seçin
2. Kağıt boyutunu seçin (A4, A3, A2, A1)
3. Ölçeği belirleyin (örn. 1:50, 1:100)
4. **Dışa Aktar**'a tıklayın

### PDF Ayarları

| Ayar | Seçenekler |
|------|-----------|
| Kağıt Boyutu | A4 / A3 / A2 / A1 |
| Yönelim | Dikey / Yatay |
| Ölçek | 1:50 / 1:100 / 1:200 / Sayfaya Sığdır |
| Çizgi Kalınlığı | İnce / Normal / Kalın |

## Yazdırma

Doğrudan yazıcıya göndermek için:

1. `CTRL + P` tuşlarına basın
2. Yazıcıyı ve kağıt boyutunu seçin
3. **Yazdır**'a tıklayın
